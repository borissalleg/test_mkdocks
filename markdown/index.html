    Guía de Markdown: Categorías y Funcionalidades

    Markdown es un lenguaje de marcado ligero que permite formatear texto sin necesidad de herramientas complejas.

    A continuación, se detallan las categorías principales y sus funcionalidades.

    Markdown

    permite aplicar formato básico al texto, como negrita, cursiva, tachado, etc.

    Negrita 
    **Negrita:** Usa doble asterisco (** ) o doble guion bajo (__).
    Ejemplo

    Este es un texto en **negrita**

    El resultado es negrita.

    Cursiva

    Cursiva : Usa un asterisco (*) o un guion bajo (_)

    Ejemplo

    Este es un texto en negrita y cursiva.

    Tachado

    Tachado: Usa dos tildes (~~).

    Ejemplo

    Este es un texto ~~tachado~~

    El resultado es ~~tachado~~

    Note

    Los encabezados permiten organizar el contenido jerárquicamente.

    Funcionalidades: Usa almohadillas (#) para definir niveles de encabezados

    Título principal (H1)-

    # Título principal

    Subtítulo (H2)

    ## Subtítulo

    Sección (H3)

    ### Sección

    Subsección (H4)

    #### Subsección
    Subsubsección (H5)
    ##### Subsubsección

    Note

    Las listas son útiles para organizar información en puntos o pasos numerados.

    Usa asteriscos (*), guiones (-) o signos de suma (+)
    • Elemento 1
    • Elemento 2
      • Subelemento 2.1
      • Subelemento 2.2
    Usa números seguidos de un punto (.)
    1. Primer paso
    2. Segundo paso
      1. Subpaso 2.1
      2. Subpaso 2.2

    Tabuladores con contenido

    Permiten incluir contenido en una nota

    nota

    se escribre asi!!!note "nota"

    warning

    se escribe warning

    danger

    se escribe danger !!!danger

    info

    se escribe info !!!info

    tip

    se escribe tip !!!tip

    example

    se escribe example !!!example

    Este es el contenido de la nota principal.

    Advertencia anidada

    Esta es una advertencia dentro de la nota principal.
